Overview

AI Blog Automator turns WordPress into an AI-assisted publishing engine: long-form articles from a topic and keywords, optional bulk queue, stock images (Pexels / Unsplash), internal links, FAQ block, SEO meta for Yoast / Rank Math / AIOSEO (or native meta), optional Article + FAQ schema, and optional Google Indexing API. Default LLM is Google Gemini; OpenAI, Anthropic Claude, and custom OpenAI-compatible APIs are supported with an Auto failover mode.

Who is it for?

  • Agencies and niche site owners who need many drafts without pasting into ChatGPT manually.
  • Publishers who want queue + cron instead of babysitting each post.
  • SEO-focused teams using Rank Math, Yoast, or AIOSEO who want titles, meta, and keywords written into those plugins.

Key features

  • Generate now — One-off AJAX generation: topic, primary keyword, optional secondaries, categories.
  • Queue — Single rows or bulk paste (tabs or pipes): title, focus keyphrase, other keywords, category IDs, article format; optional stagger by day/week.
  • LLM providers — Gemini (default), OpenAI, Claude, custom base URL + model; Auto chains providers on rate limits.
  • Images — Featured + in-post figures via official Pexels / Unsplash APIs (not deprecated Unsplash Source).
  • Internal links — Placeholders replaced with real links to published posts and pages.
  • SEO — Meta title & description, focus keyword(s), secondary keyword mapping where the SEO plugin supports it.
  • Schema — Optional JSON-LD Article and FAQ from generated content.
  • Indexing — Optional Google Indexing API with service account JSON.
  • Block editor — Generated HTML is converted to Gutenberg block comments for easier editing.
  • Activity logs — Export CSV; retention option.
  • Premium tier — Optional unlock code raises safe limits (word count headroom, images, internal links, retries, log retention). Purchase and codes are handled on the author product site.
  • Upgrade & Feedback — Built-in admin pages and welcome notice for onboarding.
  • Marketing shortcode — [aiba_product_demo] for your public product / landing page.

Requirements

  • WordPress 6.0+
  • PHP 8.0+
  • At least one LLM API key (Gemini recommended)
  • For images: Pexels and/or Unsplash API keys (optional but recommended)

Installation (summary)

  1. Upload the plugin zip via Plugins → Add New → Upload.
  2. Activate AI Blog Automator.
  3. Open AI Automator → Settings → API, add keys, Save, then Test API connections.
  4. Read docs/USER-GUIDE.html in the plugin folder (or the PDF/HTML you attach in Envato documentation).
