# Envato / CodeCanyon — Author pack for **AI Blog Automator**

Use this folder when preparing your **CodeCanyon WordPress plugin** submission. Everything here is **plain documentation** (not loaded by WordPress at runtime).

## Files in this pack

| File | Purpose |
|------|--------|
| **ITEM-DESCRIPTION.html** | Paste into the Envato item description (edit branding URLs if needed). |
| **INSTALLATION.md** | Buyer installation & first-run steps (also useful in “Documentation” tab). |
| **CHANGELOG-BUYERS.md** | Buyer-facing changelog; keep in sync with plugin releases. |
| **CREDITS-THIRD-PARTY.md** | Required third-party & API credits for reviewers. |
| **PRIVACY-DATA.md** | What data leaves the site (API keys, LLM requests, images). |
| **SUPPORT-POLICY.md** | What you support vs hosting / API limits. |
| **SCREENSHOTS-CHECKLIST.md** | Exact screens to capture for the listing (590×300 etc. per Envato current rules). |
| **VIDEO-DEMO-SCRIPT.md** | Optional voiceover / screen-recording script. |
| **SUBMISSION-CHECKLIST.md** | Pre-upload QA list. |
| **BUILD-ZIP.md** | How to build the upload zip (exclude dev files). |
| **THUMBNAIL-NOTES.md** | Using `assets/images/icon-256.png` + designing the 590×300 preview. |

## Product URL

Default: `https://devigontech.com/ai-blog-automator`  
Overridable in WordPress via `wp-config.php`: `define( 'AIBA_PRODUCT_URL', 'https://…' );`

## In-plugin helpers

- **Upgrade** admin screen: explains free vs premium and links to purchase.
- **Feedback** admin screen: messages to admin email + local inbox option.
- **Shortcode** `[aiba_product_demo]` for your marketing page.
- **Welcome notice** after activation (per-user dismiss).

## Next steps for you

1. Capture screenshots per **SCREENSHOTS-CHECKLIST.md** (or hire a designer using that list).
2. Record a short demo using **VIDEO-DEMO-SCRIPT.md** (optional but increases sales).
3. Paste **ITEM-DESCRIPTION.html** into Envato (switch to HTML mode if needed).
4. Build zip per **BUILD-ZIP.md** and run **SUBMISSION-CHECKLIST.md**.

Envato’s official rules change; always confirm the latest **WordPress Plugin Requirements** on Envato Author Help before submitting.
