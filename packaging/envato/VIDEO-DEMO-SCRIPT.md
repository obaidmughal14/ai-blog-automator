# Video demo script (~2–4 minutes)

Use for YouTube / Envato preview video. Record at 1920×1080, show real WordPress admin.

## 0:00 — Hook

“Here’s how to go from **topic + keyword** to a **draft WordPress post** with **SEO**, **images**, and **internal links**—using **AI Blog Automator**.”

## 0:15 — Install (optional cut if long)

“Upload and activate the plugin. Open **AI Automator** in the sidebar.”

## 0:30 — Settings

“Under **Settings → API**, I add my **Gemini** key—and optional **Pexels** for images. **Save**, then **Test API connections** so I know everything works.”

## 0:55 — Generate

“**Generate now**: I enter a topic and primary keyword, optional secondaries, then **Generate**. The plugin builds outline, sections, FAQ, images, links, and SEO meta.”

## 1:25 — Result

“Opening the new **draft**: block editor sections, featured image, and Rank Math / Yoast fields filled where configured.”

## 1:50 — Queue (optional)

“For scale, **Queue** accepts bulk lines from a spreadsheet—great for a month of content.”

## 2:10 — Premium / outro

“**Upgrade** explains optional premium limits. Links and docs are in the plugin. Thanks for watching—grab it on CodeCanyon.”
